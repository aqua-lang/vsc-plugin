# :droplet: Aqua

Visual Studio Code support for Aqua programming language, featuring syntax highlighting, auto-completing, formatting, definition looking up, and other useful tools.

> **Note: This plugin is currently under active development, so expect bugs in usage.**

In order for this plugin to actually work in your workspace, you need to install the Aqua compiler through [~~Here~~](https://www.youtube.com/watch?v=dQw4w9WgXcQ).

## Contribution

This project absolutely welcomes contributions! However, please note that **any code SHOULD be written by hand, and any code generated by coding agents MUST be reviewed and remastered**.
